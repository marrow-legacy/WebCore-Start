You can have your newly created project automatically checked into an SCM system like Subversion, Git, or Mercurial.

Select your source code manager.
The URL to your repository. Examples:
http://example.com/HelloWorld/trunk
file:///repository/HelloWorld/trunk
https://jrh@bitbucket.org/jrh/HelloWorld/
git@github.com:jrh/HelloWorld.git
Leave blank if the URL already includes a user name. Leave both User Name and Password blank if you do not require authentication.
Leave blank if authentication is automatic, e.g. if you have a configuration file which already contains your password.
